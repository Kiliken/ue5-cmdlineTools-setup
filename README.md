# ue5-cmdlineTools-setup
A minimal setup for building [UE5](https://www.unrealengine.com/en-US/unreal-engine-5) projects using command line tools.

```powershell
curl.exe -L -o "$(Get-Location)\codeDL.ps1" "https://raw.githubusercontent.com/Kiliken/ue5-cmdlineTools-setup/main/codeDL.ps1"
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "$(Get-Location)\codeDL.ps1"

```
